# Fish Pond Chatbot Frontend

A minimal React frontend for the Fish Pond AI Chatbot.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser.

## Backend Integration

This frontend communicates with the FastAPI backend running on http://localhost:8000.

Make sure the backend is running before using the chat feature.

## Features

- Chat interface to ask questions about fish pond management
- Displays AI responses with sensor data snapshots
- Simple, clean UI
